#### 问题1：Workflow 实体与 schema.sql 不一致 🟡 中等

**描述**: Workflow.java 实体类包含 `workflowConfig` 字段，但 schema.sql 中 t_workflow 表缺少该字段。

**影响**:
- 数据库表缺少 `workflow_config` 字段
- 保存工作流配置会丢失数据

**修复建议**:
在 schema.sql 的 t_workflow 表中添加 `workflow_config` 字段：

```sql
ALTER TABLE `t_workflow` ADD COLUMN `workflow_config` TEXT COMMENT '工作流配置JSON';
```
