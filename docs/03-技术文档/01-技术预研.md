# 分布式定时任务调度平台 - 技术预研文档

## 1. XXL-JOB 2.4.x 与 Spring Boot 3.x 兼容性验证

### 1.1 版本选择依据

| 组件 | 版本 | 选择理由 |
|-----|------|---------|
| Spring Boot | 3.2.x | LTS 版本，支持 Java 21，性能优化显著 |
| XXL-JOB | 2.4.1 | 最新稳定版本，支持 Spring Boot 3.x |
| Java | 21 | LTS 版本，性能和安全性提升 |

### 1.2 依赖兼容性验证

**核心依赖清单**：

| 依赖 | GroupId | ArtifactId | 版本 | 说明 |
|-----|---------|-----------|------|------|
| Spring Boot Starter | org.springframework.boot | spring-boot-starter | 3.2.x | 核心依赖 |
| Spring Boot Starter Web | org.springframework.boot | spring-boot-starter-web | 3.2.x | Web 支持 |
| XXL-JOB Core | com.xuxueli | xxl-job-core | 2.4.1 | 核心调度组件 |
| XXL-JOB Admin | com.xuxueli | xxl-job-admin | 2.4.1 | 调度中心 |
| MyBatis-Plus | com.baomidou | mybatis-plus-spring-boot3-starter | 3.5.5 | ORM 框架 |
| MySQL Connector | com.mysql | mysql-connector-j | 8.3.0 | 数据库驱动 |
| Redis Starter | org.springframework.boot | spring-boot-starter-data-redis | 3.2.x | Redis 支持 |
| Redisson | org.redisson | redisson-spring-boot-starter | 3.23.3 | 分布式锁 |
| RabbitMQ Starter | org.springframework.boot | spring-boot-starter-amqp | 3.2.x | 消息队列 |

### 1.3 兼容性测试要点

| 测试项 | 测试方法 | 预期结果 |
|-------|---------|---------|
| 项目启动 | 启动 Spring Boot 3.x + XXL-JOB 2.4.x | 正常启动，无依赖冲突 |
| 任务注册 | 创建并注册定时任务 | 任务成功注册到调度中心 |
| 任务执行 | 触发任务执行 | 执行器正常接收并执行任务 |

| 分片执行 | 创建分片任务 | 多个执行器正确分配分片 |

### 1.4 已知兼容性问题与解决方案

| 问题 | 描述 | 解决方案 |
|-----|------|---------|
| Jackson 版本冲突 | Spring Boot 3.x 使用 Jackson 2.15+ | XXL-JOB 2.4.x 已兼容 |
| Jakarta EE API | Spring Boot 3.x 使用 Jakarta EE 9+ | XXL-JOB 2.4.x 已迁移 |
| JAXB 依赖 | Java 11+ 移除了 JAXB | 手动添加 JAXB 依赖 |

## 2. 性能基准测试方案

### 2.1 测试指标

| 指标 | 目标值 | 测试方法 |
|-----|-------|---------|
| 调度响应时间 | ≤ 100ms | 压测任务触发延迟 |
| 告警延迟 | ≤ 10秒 | 模拟任务失败，记录告警触发时间 |
| 任务吞吐量 | ≥ 1000 任务/分钟 | 压测系统处理能力 |
| 并发执行能力 | ≥ 100 并发任务 | 多任务同时执行 |

## 4. 高可用方案设计

### 4.1 调度中心高可用

| 方案 | 说明 | 实现方式 |
|-----|------|---------|
| 多节点部署 | 部署多个 Admin 节点 | 通过数据库锁实现选举 |
| 负载均衡 | Nginx 反向代理 | 轮询策略 |

### 4.2 执行器高可用

| 方案 | 说明 | 实现方式 |
|-----|------|---------|
| 集群部署 | 部署多个执行器节点 | 自动注册和发现 |
| 故障转移 | 执行器离线时自动切换 | 心跳检测 + 自动剔除 |

### 4.3 数据存储高可用

| 组件 | 方案 | 说明 |
|-----|------|------|
| MySQL | 主从复制 | 读写分离 |
| Redis | 哨兵模式 | 自动故障转移 |
| RabbitMQ | 镜像队列 | 消息持久化 |
