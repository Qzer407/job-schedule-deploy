# 监控中心模块设计

## 1. 模块概述

监控中心模块负责系统的运行状态监控，包括任务执行统计、执行器健康检查、执行日志查询等功能，为运维人员提供全面的系统监控视图。

## 2. 功能设计

### 2.1 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 执行统计 | 任务执行成功率、失败率统计 | P0 |
| 执行器状态 | 查看执行器在线状态和健康检查 | P0 |
| 任务日志 | 查询任务执行日志详情 | P1 |
| 仪表盘 | 汇总统计信息展示 | P1 |
| 趋势图表 | 任务执行趋势分析 | P2 |

### 2.2 核心业务流程

```
监控统计请求
       │
       ▼
┌─────────────────┐     ┌──────────────┐     ┌─────────────┐
│ MonitorController│ ──►│ MonitorService│ ──►│TaskLogMapper│
│                 │     │ 统计计算     │     │ 日志查询    │
└────────┬────────┘     └──────┬───────┘     └─────────────┘
         │                     │
         │                     ▼
         │              ┌─────────────┐
         │              │ExecutorService│
         │              │ 执行器检查   │
         │              └─────────────┘
         │
         ▼
┌─────────────────┐
│  返回统计数据    │
└─────────────────┘
```

## 4. API接口设计

### 4.1 接口列表

| HTTP方法 | API路径 | 功能描述 | Controller文件 |
|----------|---------|----------|----------------|
| GET | `/api/v1/monitor/statistics` | 获取执行统计 | MonitorController |
| GET | `/api/v1/monitor/executors` | 获取执行器状态 | MonitorController |
| GET | `/api/v1/logs` | 查询任务日志 | TaskLogController |
| GET | `/api/v1/logs/{id}` | 查询日志详情 | TaskLogController |
| DELETE | `/api/v1/logs/{id}` | 删除日志 | TaskLogController |
| POST | `/api/v1/logs/clean` | 清理日志 | TaskLogController |

### 4.2 请求响应示例

**获取执行统计:**

}

## 5. 数据库设计

### 5.1 核心数据表

#### t_executor_info（执行器信息表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY AUTO_INCREMENT | 执行器ID |
| executor_name | VARCHAR(200) | NOT NULL | 执行器名称 |
| executor_address | VARCHAR(200) | NOT NULL | 执行器地址 |
| executor_group | VARCHAR(100) | NOT NULL | 执行器分组 |
| status | TINYINT | DEFAULT 1 | 状态(0:离线,1:在线) |
| last_heartbeat | DATETIME | | 最后心跳时间 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

#### t_task_log（任务日志表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY AUTO_INCREMENT | 日志ID |
| task_id | BIGINT | NOT NULL | 任务ID |
| trigger_time | DATETIME | NOT NULL | 触发时间 |
| start_time | DATETIME | | 开始时间 |
| end_time | DATETIME | | 结束时间 |
| execution_status | TINYINT | DEFAULT 0 | 执行状态(0:待执行,1:成功,2:失败) |
| execution_log | TEXT | | 执行日志 |
| error_msg | TEXT | | 错误信息 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
