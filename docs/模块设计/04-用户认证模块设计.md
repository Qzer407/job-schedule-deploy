# 用户认证模块设计

## 1. 模块概述

用户认证模块负责系统的用户管理和权限控制，基于Spring Security实现JWT认证，支持用户登录、注册、角色权限控制等功能。

## 2. 功能设计

### 2.1 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 用户登录 | 用户名密码登录，返回JWT Token | P0 |
| 用户注册 | 创建新用户账号 | P0 |
| 用户登出 | 清除Token，退出登录 | P0 |
| 用户信息查询 | 查询当前登录用户信息 | P1 |
| 权限控制 | 基于角色的接口访问控制 | P0 |
| Token刷新 | 刷新过期Token | P1 |

### 2.2 核心业务流程

```
用户登录请求
       │
       ▼
┌─────────────────┐     ┌──────────────┐     ┌─────────────┐
│ AuthController  │ ──► │ UserService  │ ──► │UserMapper   │
│ 验证登录信息    │     │ 验证密码     │     │ 查询用户    │
└────────┬────────┘     └──────┬───────┘     └─────────────┘
         │                     │
         │                     ▼
         │              ┌─────────────┐
         │              │ BCrypt验证  │
         │              └──────┬──────┘
         │                     │
         │                     ▼
         │              ┌─────────────┐
         │              │生成JWT Token│
         │              └──────┬──────┘
         │                     │
         ▼                     ▼
┌─────────────────┐     ┌─────────────┐
│  返回Token      │     │ 存储到响应头│
└─────────────────┘     └─────────────┘
```

## 4. API接口设计

### 4.1 接口列表

| HTTP方法 | API路径 | 功能描述 | Controller文件 |
|----------|---------|----------|----------------|
| POST | `/api/v1/auth/login` | 用户登录 | AuthController |
| POST | `/api/v1/auth/register` | 用户注册 | AuthController |
| POST | `/api/v1/auth/logout` | 用户登出 | AuthController |
| GET | `/api/v1/auth/me` | 获取当前用户信息 | AuthController |
| POST | `/api/v1/auth/refresh` | 刷新Token | AuthController |

## 5. 数据库设计

### 5.1 核心数据表

#### t_user（用户表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY AUTO_INCREMENT | 用户ID |
| username | VARCHAR(100) | NOT NULL UNIQUE | 用户名 |
| password | VARCHAR(200) | NOT NULL | 密码(BCrypt加密) |
| email | VARCHAR(200) | | 邮箱 |
| role | VARCHAR(50) | DEFAULT 'USER' | 角色(ADMIN/USER/GUEST) |
| status | TINYINT | DEFAULT 1 | 状态(0:禁用,1:启用) |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
