# 工作流编排模块设计

## 1. 模块概述

工作流编排模块支持任务依赖编排，实现复杂的DAG（有向无环图）工作流，允许用户通过可视化编辑器创建任务依赖关系，并按依赖顺序执行任务。

## 2. 功能设计

### 2.1 功能列表

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 工作流管理 | 创建、编辑、删除工作流 | P0 |
| DAG可视化编辑 | 可视化创建任务依赖关系 | P0 |
| 工作流执行 | 按依赖顺序执行工作流 | P0 |
| 工作流状态查询 | 查询工作流执行状态 | P1 |
| 工作流暂停/终止 | 暂停或终止正在执行的工作流 | P1 |
| 任务依赖配置 | 配置任务间的依赖关系 | P0 |

### 2.2 核心业务流程

```
创建工作流
    │
    ▼
┌─────────────┐     ┌──────────────┐     ┌─────────────────┐
│ Workflow    │ ──► │ Workflow     │ ──► │ DagTopologySort │
│ Controller  │     │ ServiceImpl  │     │   (拓扑排序)     │
└─────────────┘     └──────────────┘     └────────┬────────┘
                                                   │
                                                   ▼
                                       ┌─────────────────┐
                                       │WorkflowExecutor │
                                       │  (工作流执行器) │
                                       └────────┬────────┘
                                                │
                        ┌───────────────────────┼───────────────────────┐
                        ▼                       ▼                       ▼
              ┌───────────────┐      ┌───────────────┐      ┌───────────────┐
              │ 任务A执行完成  │ ──► │ 任务B执行完成  │ ──► │ 任务C执行完成  │
              └───────────────┘      └───────────────┘      └───────────────┘
```

| GET | `/api/v1/workflows/{id}/dependencies` | 查询工作流依赖 | TaskDependencyController |

## 5. 数据库设计

### 5.1 核心数据表

#### t_workflow（工作流表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY AUTO_INCREMENT | 工作流ID |
| workflow_name | VARCHAR(200) | NOT NULL | 工作流名称 |
| workflow_desc | VARCHAR(500) | | 工作流描述 |
| status | TINYINT | DEFAULT 0 | 状态(0:停用,1:启用) |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

#### t_workflow_execution（工作流执行记录表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY AUTO_INCREMENT | 执行记录ID |
| workflow_id | BIGINT | NOT NULL | 工作流ID |
| execution_status | TINYINT | DEFAULT 0 | 执行状态(0:待执行,1:执行中,2:成功,3:失败) |
| trigger_type | VARCHAR(50) | | 触发类型(MANUAL/AUTO) |
| start_time | DATETIME | | 开始时间 |
| end_time | DATETIME | | 结束时间 |
| execution_log | TEXT | | 执行日志 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

#### t_task_dependency（任务依赖表）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY AUTO_INCREMENT | 依赖ID |
| workflow_id | BIGINT | NOT NULL | 工作流ID |
| task_id | BIGINT | NOT NULL | 任务ID |
| parent_task_id | BIGINT | | 父任务ID |
| dependency_type | VARCHAR(50) | DEFAULT 'SUCCESS' | 依赖类型(SUCCESS/FAILURE/ALWAYS) |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
