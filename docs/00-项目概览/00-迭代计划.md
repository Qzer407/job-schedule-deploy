# 项目迭代计划

## 一、项目概述

### 1.1 项目名称
**QzerSchedule** - 分布式定时任务调度平台

### 1.2 项目定位
基于 XXL-JOB 的分布式任务调度系统，提供可视化任务管理、实时监控、多渠道告警等功能。

### 1.3 技术栈
- **后端**：Spring Boot 3.2 + MyBatis-Plus 3.5 + XXL-JOB 2.4
- **前端**：Vue 3 + Element Plus + Vite
- **数据库**：MySQL 8.0
- **部署**：Docker + Docker Compose
